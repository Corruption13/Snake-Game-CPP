# Snake-Game-CPP
Snake Game designed and implemented from scratch in under 2 hours limit, without reference, as random challenge for myself.


## Requirements:

	$ sudo apt-get install libncurses-dev

## Compile Using:

	$ g++ snake.cpp -o snake.exe -lncurses

## Execute:

	$ ./snake.exe
